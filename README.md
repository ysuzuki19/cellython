# Cellython (under development)
python script with cell.


## Get Start

```bash
$ git clone https://github.com/ysuzuki19/cellython
```

## Run server

```bash
$ cd api
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install flask python-dotenv
$ flask run
```

## Run frontend

```bash
$ cd frontend
$ npm start
```
